# apod_downloader

This is the C++ version of my nasa picture of the day wallpaper downloader. It's not yet complete. Use my Rust or NodeJS versions in the interim.

## dependencies
Boost 
folly 
Threads  
gflags  
curlpp  

## to build
```
cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug -G "Unix Makefiles"
```
